## Project

The Entity Consumer is a NestJS-based microservice designed to consume Kafka messages published by the Entity Producer service. It leverages Protobuf for efficient, strongly-typed message serialization and deserialization, using the protobuf-ts library to decode incoming messages.

This service listens to Kafka topics for entity-related events (e.g., entity-created), processes these events according to business logic, and integrates with other parts of your system as needed.

By using Kafka as a message broker and Protobuf for schema validation, this consumer ensures robust, scalable, and maintainable event-driven architecture communication.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Protobuf Code Generation

Both projects use buf v2 for managing Protobuf schemas and code generation.

Make sure you have buf installed, then run:

```bash
buf generate
```

in both entity-producer and entity-consumer directories.

## Environment Variables

Configure `.env` (example):

```env
KAKFA_ENDPOINT=localhost:9092
```
